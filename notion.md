白天 Python 学完后，留 1 分钟写个 “C++ 转换笔记”：比如今天用 Python 写了回溯法的 “组合总和”，随手记一句 “Python 的 list 选元素→C++ 用 vector，剪枝条件要注意传引用减少拷贝”，晚上复现的时候直接看笔记，不用重新想 “Python 代码怎么转 C++”，节省时间。
晚上复现 C++ 时，重点练 “手写 + 口述”：写代码的时候，同步小声说 “这里用 unordered_map 存前缀和，因为嵌入式里哈希表比 map 快”“这个链表节点指针要先判空，避免野指针”—— 既练了代码，又提前准备了面试时的 “代码讲解话术”（嵌入式面试常要求边写边讲思路）。
